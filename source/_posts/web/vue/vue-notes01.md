## 安装Nodejs

首先安装Nodejs环境，参考我的博客《Nodejs的安装和配置》即可。

## 通过Vite脚本初始化
```
cnpm init vite@latest
cnpm install vuex@next
cnpm install vue-router@4
```
项目开发执行命令
```
cnpm install
cnpm run dev  #启动开发服务器
cnpm run build #为生产环境构建
cnpm run serve #本地预览
```
