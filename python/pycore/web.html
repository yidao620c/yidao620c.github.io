pythonТаИт┐Ѓ - webт╝ђтЈЉ

тЈЉУАеС║ј 2015-12-30 тѕєу▒╗С║ј python жўЁУ»╗ТгАТЋ░№╝џ

Webт║ћућет╝ђтЈЉтЈ»С╗ЦУ»┤Тў»уЏ«тЅЇУй»С╗Хт╝ђтЈЉСИГТюђжЄЇУдЂуџёжЃетѕєсђѓWebт╝ђтЈЉС╣Ъу╗ЈтјєС║єтЦйтЄаСИфжўХТ«х№╝џжЮЎТђЂWebжАхжЮбсђЂCGIсђЂASP/JSP/PHPсђЂMVCсђѓ

уЏ«тЅЇ№╝їWebт╝ђтЈЉТіђТю»С╗Їтюет┐ФжђЪтЈЉт▒ЋСИГ№╝їт╝ѓТГЦт╝ђтЈЉсђЂТќ░уџёMVVMтЅЇуФ»ТіђТю»т▒ѓтЄ║СИЇуЕисђѓ

PythonуџёУ»ъућЪтјєтЈ▓Т»ћWebУ┐ўУдЂТЌЕ№╝їућ▒С║јPythonТў»СИђуДЇУДБжЄітъІуџёУёџТюгУ»ГУеђ№╝їт╝ђтЈЉТЋѕујЄжФў№╝їТЅђС╗ЦжЮътИИжђѓтљѕућеТЮЦтЂџWebт╝ђтЈЉсђѓ

PythonТюЅСИіуЎЙуДЇWebт╝ђтЈЉТАєТъХ№╝їТюЅтЙѕтцџТѕљуєЪуџёТеАТЮ┐ТіђТю»№╝їжђЅТІЕPythonт╝ђтЈЉWebт║ћуће№╝їСИЇСйєт╝ђтЈЉТЋѕујЄжФў№╝їУђїСИћУ┐љУАїжђЪт║дт┐Фсђѓ

WSGIТјЦтЈБ

СИђСИфWebт║ћућеуџёТюгУ┤ет░▒Тў»

  1. ТхЈУДѕтЎетЈЉжђЂСИђСИфHTTPУ»иТ▒ѓ№╝Џ
  2. ТюЇтіАтЎеТћХтѕ░У»иТ▒ѓ№╝їућЪТѕљСИђСИфHTMLТќЄТАБ№╝Џ
  3. ТюЇтіАтЎеТііHTMLТќЄТАБСйюСИ║HTTPтЊЇт║ћуџёBodyтЈЉжђЂу╗ЎТхЈУДѕтЎе№╝Џ
  4. ТхЈУДѕтЎеТћХтѕ░HTTPтЊЇт║ћ№╝їС╗јHTTP BodyтЈќтЄ║HTMLТќЄТАБт╣ХТўЙуц║сђѓ

ТЅђС╗Ц№╝їТюђу«ђтЇЋуџёWebт║ћућет░▒Тў»тЁѕТііHTMLућеТќЄС╗ХС┐ЮтГўтЦй№╝їућеСИђСИфуј░ТѕљуџёHTTPТюЇтіАтЎеУй»С╗Х№╝їТјЦТћХућеТѕиУ»иТ▒ѓ№╝їС╗јТќЄС╗ХСИГУ»╗тЈќHTMLУ┐ћтЏъсђѓ ApacheсђЂNginxсђЂLighttpdуГЅУ┐ЎС║ЏтИИУДЂуџёжЮЎТђЂТюЇтіАтЎет░▒Тў»т╣▓У┐ЎС╗ХС║ІТЃЁуџёсђѓ

тдѓТъюУдЂтіеТђЂућЪТѕљHTML№╝їт░▒жюђУдЂТііСИіУ┐░ТГЦжфцУЄфти▒ТЮЦт«ъуј░сђѓСИЇУ┐Є№╝їТјЦтЈЌHTTPУ»иТ▒ѓсђЂУДБТъљHTTPУ»иТ▒ѓсђЂтЈЉжђЂHTTPтЊЇт║ћжЃйТў»УІдтіЏТ┤╗№╝ї тдѓТъюТѕЉС╗гУЄфти▒ТЮЦтєЎУ┐ЎС║Џт║Ћт▒ѓС╗БуаЂ№╝їУ┐ўТ▓Ат╝ђтДІтєЎтіеТђЂHTMLтЉб№╝їт░▒тЙЌУі▒СИфТііТюѕтј╗У»╗HTTPУДёУїЃсђѓ

ТГБуА«уџётЂџТ│ЋТў»т║Ћт▒ѓС╗БуаЂућ▒СИЊжЌеуџёТюЇтіАтЎеУй»С╗Хт«ъуј░№╝їТѕЉС╗гућеPythonСИЊТ│еС║јућЪТѕљHTMLТќЄТАБсђѓ тЏаСИ║ТѕЉС╗гСИЇтИїТюЏТјЦУДдтѕ░TCPУ┐ъТјЦсђЂHTTPтјЪтДІУ»иТ▒ѓтњїтЊЇт║ћТа╝т╝Ј№╝їТЅђС╗Ц№╝їжюђУдЂСИђСИфу╗ЪСИђуџёТјЦтЈБ№╝їУ«ЕТѕЉС╗гСИЊт┐ЃућеPythonу╝ќтєЎWebСИџтіАсђѓ

У┐ЎСИфТјЦтЈБт░▒Тў»WSGI№╝џWeb Server Gateway Interfaceсђѓ

тЁ│С║јWSGIТѕЉТюЅСИЊжЌеуџёТќЄуФаУ«▓УДБ№╝їУ┐ЎжЄїт░▒СИЇжЄЇтцЇС║є№╝їТёЪтЁ┤УХБуџётЈ»С╗Цтј╗у┐╗у┐╗сђѓ

WebТАєТъХ

тЁХт«ъСИђСИфWeb App№╝їт░▒Тў»тєЎСИђСИфWSGIуџётцёуљєтЄйТЋ░№╝їжњѕт»╣Т»ЈСИфHTTPУ»иТ▒ѓУ┐ЏУАїтЊЇт║ћсђѓ СйєТў»уЏ┤ТјЦУЄфти▒у╝ќтєЎWSGIтцёуљєтЄйТЋ░С╝џТўЙтЙЌжЮътИИу╣Ђуљљ№╝їУ┐ЎжЄїт«їтЁетЈ»С╗ЦућеТАєТъХС╗БТЏ┐ТѕЉС╗гтЂџсђѓ

ућ▒С║јућеPythonт╝ђтЈЉСИђСИфWebТАєТъХтЇЂтѕєт«╣ТўЊ№╝їТЅђС╗ЦPythonТюЅСИіуЎЙСИфт╝ђТ║љуџёWebТАєТъХсђѓ У┐ЎжЄїТѕЉС╗гтЁѕСИЇУ«еУ«║тљёуДЇWebТАєТъХуџёС╝ўу╝║уѓ╣№╝їуЏ┤ТјЦжђЅТІЕСИђСИфТ»ћУЙЃТхЂУАїуџёWebТАєТъХРђћРђћFlaskТЮЦСй┐ућесђѓ

ућеFlaskу╝ќтєЎWeb AppТ»ћWSGIТјЦтЈБу«ђтЇЋ№╝ѕУ┐ЎСИЇТў»т║ЪУ»ЮС╣ѕ№╝їУдЂТў»Т»ћWSGIУ┐ўтцЇТЮѓ№╝їућеТАєТъХт╣▓тўЏ№╝Ъ№╝Ѕ№╝їТѕЉС╗гтЁѕућеpipт«ЅУБЁFlask№╝џ

уёХтљјтєЎСИђСИфapp.py№╝їтцёуљє3СИфURL№╝їтѕєтѕФТў»№╝џ

  • GET /№╝џждќжАх№╝їУ┐ћтЏъHome№╝Џ
  • GET /signin№╝џуЎ╗тйЋжАх№╝їТўЙуц║уЎ╗тйЋУАетЇЋ№╝Џ
  • POST /signin№╝џтцёуљєуЎ╗тйЋУАетЇЋ№╝їТўЙуц║уЎ╗тйЋу╗ЊТъюсђѓ

Т│еТёЈСИіжЮб№╝їтљїСИђСИфURL/signinтѕєтѕФТюЅGETтњїPOSTСИцуДЇУ»иТ▒ѓ№╝їТўат░ётѕ░СИцСИфтцёуљєтЄйТЋ░СИГсђѓ

FlaskжђџУ┐ЄPythonуџёУБЁжЦ░тЎетюетєЁжЃеУЄфтіетю░ТііURLтњїтЄйТЋ░у╗ЎтЁ│УЂћУхиТЮЦ№╝їТЅђС╗Ц№╝їТѕЉС╗гтєЎтЄ║ТЮЦуџёС╗БуаЂт░▒тЃЈУ┐ЎТаи№╝џ

У┐љУАїpython app.py№╝їFlaskУЄфтИдуџёServerтюеуФ»тЈБ5000СИіуЏЉтљг№╝ї ТЅЊт╝ђТхЈУДѕтЎе№╝їУЙЊтЁЦждќжАхтю░тЮђ http://localhost:5000/ уюІуюІТЋѕТъюсђѓ

тєЇтюеТхЈУДѕтЎетю░тЮђТаЈУЙЊтЁЦhttp://localhost:5000/signin№╝їС╝џТўЙуц║уЎ╗тйЋУАетЇЋсђѓ УЙЊтЁЦжбёУ«ЙуџёућеТѕитљЇadminтњїтЈБС╗цpassword№╝їуЎ╗тйЋТѕљтіЪ№╝їУЙЊтЁЦтЁХС╗ќт░▒ТўЙуц║уЎ╗жЎєтц▒У┤Цсђѓ

т«ъжЎЁуџёWeb Appт║ћУ»ЦТІ┐тѕ░ућеТѕитљЇтњїтЈБС╗цтљј№╝їтј╗ТЋ░ТЇ«т║ЊТЪЦУ»бтєЇТ»ћт»╣№╝їТЮЦтѕцТќГућеТѕиТў»тљдУЃйуЎ╗тйЋТѕљтіЪсђѓ

жЎцС║єFlask№╝їтИИУДЂуџёPython WebТАєТъХУ┐ўТюЅ№╝џ

  • Django№╝џтЁеУЃйтъІWebТАєТъХ№╝Џ
  • web.py№╝џСИђСИфт░ЈтиДуџёWebТАєТъХ№╝Џ
  • Bottle№╝џтњїFlaskу▒╗С╝╝уџёWebТАєТъХ№╝Џ
  • Tornado№╝џFacebookуџёт╝ђТ║љт╝ѓТГЦWebТАєТъХсђѓ

ТюЅС║єWebТАєТъХ№╝їТѕЉС╗гтюеу╝ќтєЎWebт║ћућеТЌХ№╝їТ│еТёЈтіЏт░▒С╗јWSGIтцёуљєтЄйТЋ░УйгуД╗тѕ░URL+т»╣т║ћуџётцёуљєтЄйТЋ░№╝їУ┐ЎТаи№╝їу╝ќтєЎWeb Appт░▒ТЏ┤тіау«ђтЇЋС║єсђѓ

Сй┐ућеТеАТЮ┐

тюетЄйТЋ░СИГУ┐ћтЏъСИђСИфтїЁтљФHTMLуџётГЌугдСИ▓№╝їу«ђтЇЋуџёжАхжЮбУ┐ўтЈ»С╗Ц№╝ї СйєТў»№╝їСИђС║ЏтцЇТЮѓуџёwebт║ћућеТѕљуЎЙСИітЇЃСИфHTMLжАхжЮб№╝їСйауА«С┐АУЃйтюеPythonуџётГЌугдСИ▓СИГТГБуА«тю░тєЎтЄ║ТЮЦС╣ѕ№╝ЪтЈЇТГБТѕЉТў»тЂџСИЇтѕ░сђѓ

ТюЅWebт╝ђтЈЉу╗ЈжфїуџётљїтГджЃйТўјуЎй№╝їWeb AppТюђтцЇТЮѓуџёжЃетѕєт░▒тюеHTMLжАхжЮбсђѓHTMLСИЇС╗ЁУдЂТГБуА«№╝їУ┐ўУдЂжђџУ┐ЄCSSуЙјтїќ№╝ї тєЇтіаСИітцЇТЮѓуџёJavaScriptУёџТюгТЮЦт«ъуј░тљёуДЇС║цС║њтњїтіеућ╗ТЋѕТъюсђѓТђ╗С╣І№╝їућЪТѕљHTMLжАхжЮбуџёжџЙт║дтЙѕтцДсђѓ

ућ▒С║јтюеPythonС╗БуаЂжЄїТІ╝тГЌугдСИ▓Тў»СИЇуј░т«ъуџё№╝їТЅђС╗Ц№╝їТеАТЮ┐ТіђТю»тЄ║уј░С║єсђѓ

Сй┐ућеТеАТЮ┐№╝їТѕЉС╗гжюђУдЂжбётЁѕтЄєтцЄСИђСИфHTMLТќЄТАБ№╝їУ┐ЎСИфHTMLТќЄТАБСИЇТў»ТЎ«жђџуџёHTML№╝ї УђїТў»тхїтЁЦС║єСИђС║ЏтЈўжЄЈтњїТїЄС╗ц№╝їуёХтљј№╝їТа╣ТЇ«ТѕЉС╗гС╝атЁЦуџёТЋ░ТЇ«№╝їТЏ┐ТЇбтљј№╝їтЙЌтѕ░Тюђу╗ѕуџёHTML№╝їтЈЉжђЂу╗ЎућеТѕи№╝џ У┐Ўт░▒Тў»С╝аУ»┤СИГуџёMVC№╝џModel-View-Controller№╝їСИГТќЄтљЇРђЮТеАтъІ-УДєтЏЙ-ТјДтѕХтЎеРђЮсђѓ

PythonтцёуљєURLуџётЄйТЋ░т░▒Тў»C№╝џController№╝їControllerУ┤ЪУ┤БСИџтіАжђ╗УЙЉ№╝їТ»ћтдѓТБђТЪЦућеТѕитљЇТў»тљдтГўтюе№╝їтЈќтЄ║ућеТѕиС┐АТЂ»уГЅуГЅ№╝Џ

тїЁтљФтЈўжЄЈ@@ name @@уџёТеАТЮ┐т░▒Тў»V№╝џView№╝їViewУ┤ЪУ┤БТўЙуц║жђ╗УЙЉ№╝їжђџУ┐Єу«ђтЇЋтю░ТЏ┐ТЇбСИђС║ЏтЈўжЄЈ№╝їViewТюђу╗ѕУЙЊтЄ║уџёт░▒Тў»ућеТѕиуюІтѕ░уџёHTMLсђѓ

MVCСИГуџёModelтюетЊф№╝ЪModelТў»ућеТЮЦС╝ау╗ЎViewуџё№╝їУ┐ЎТаиViewтюеТЏ┐ТЇбтЈўжЄЈуџёТЌХтђЎ№╝їт░▒тЈ»С╗ЦС╗јModelСИГтЈќтЄ║уЏИт║ћуџёТЋ░ТЇ«сђѓ

СИіжЮбуџёСЙІтГљСИГ№╝їModelт░▒Тў»СИђСИфdict№╝џ{ 'name': 'Michael' }

уј░тюе№╝їТѕЉС╗гТііСИіТгАуЏ┤ТјЦУЙЊтЄ║тГЌугдСИ▓СйюСИ║HTMLуџёСЙІтГљућежФўуФ»тцДТ░ћСИіТАБТгАуџёMVCТеАт╝ЈТћ╣тєЎСИђСИІ№╝џ

FlaskжђџУ┐Єrender_template()тЄйТЋ░ТЮЦт«ъуј░ТеАТЮ┐уџёТИ▓ТЪЊсђѓтњїWebТАєТъХу▒╗С╝╝№╝ї PythonуџёТеАТЮ┐С╣ЪТюЅтЙѕтцџуДЇсђѓFlaskж╗ўУ«цТћ»ТїЂуџёТеАТЮ┐Тў»jinja2№╝їТЅђС╗ЦТѕЉС╗гтЁѕуЏ┤ТјЦт«ЅУБЁjinja2№╝џ

уёХтљј№╝їт╝ђтДІу╝ќтєЎjinja2ждќжАхТеАТЮ┐№╝џ

ућеТЮЦТўЙуц║уЎ╗тйЋУАетЇЋуџёТеАТЮ┐№╝џ

уЎ╗тйЋТѕљтіЪуџёТеАТЮ┐№╝џ

уЎ╗тйЋтц▒У┤ЦуџёТеАТЮ┐тЉб№╝ЪТѕЉС╗гтюеform.htmlСИГтіаС║єСИђуѓ╣ТЮАС╗ХтѕцТќГ№╝їТііform.htmlжЄЇућеСИ║уЎ╗тйЋтц▒У┤ЦуџёТеАТЮ┐сђѓ

Тюђтљј№╝їСИђт«џУдЂТііТеАТЮ┐ТћЙтѕ░ТГБуА«уџёtemplatesуЏ«тйЋСИІ№╝їtemplatesтњїapp.pyтюетљїу║ДуЏ«тйЋСИІ№╝џ

тљ»тіеpython app.py№╝їуюІуюІСй┐ућеТеАТЮ┐уџёжАхжЮбТЋѕТъю№╝їСИЇжЄЇтцЇТ╝ћуц║С║є
