发表于 2016-04-30 分类于 python 阅读次数：

Django是一个非常强大的web框架，能让你快速的构建应用，它本身包含了一个简单的服务器程序，让你在开发环境里调试用。 但是在生产环境中就需要将其部署到更专业的web服务器里去了，比如Apache、Nginx等。
