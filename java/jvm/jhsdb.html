JVM工具-可视化工具JHSDB

发表于 2021-04-17 分类于 java

JHSDB是一款基于服务性代理实现的进程外调试工具。可以在一个独立的JVM进程里分析其他HotSpot虚拟机内部数据。 或者从HotSpot虚拟机进程内存中dump出来的转储快照里还原它的运行状态细节。

该工具从JDK9的时候开始提供，随JDK一起发布，无需另外下载。
