SpringBootзі»еҲ— - иҮӘе·ұеҶҷstarter

еҸ‘иЎЁдәҺ 2017-07-22 еҲҶзұ»дәҺ java йҳ…иҜ»ж¬Ўж•°пјҡ

Spring Bootз”ұдј—еӨҡStarterз»„жҲҗпјҢйҡҸзқҖзүҲжң¬зҡ„жҺЁз§»Starterе®¶ж—ҸжҲҗе‘ҳд№ҹдёҺж—ҘдҝұеўһгҖӮеңЁдј з»ҹMavenйЎ№зӣ®дёӯйҖҡеёёе°ҶдёҖдәӣеұӮгҖҒз»„д»¶жӢҶеҲҶдёәжЁЎеқ—жқҘз®ЎзҗҶпјҢ д»Ҙдҫҝзӣёдә’дҫқиө–еӨҚз”ЁпјҢеңЁSpring BootйЎ№зӣ®дёӯжҲ‘д»¬еҲҷеҸҜд»ҘеҲӣе»әиҮӘе®ҡд№үSpring Boot StarterжқҘиҫҫжҲҗиҜҘзӣ®зҡ„гҖӮ

еҸҜд»Ҙи®ӨдёәstarterжҳҜдёҖз§ҚжңҚеҠЎвҖ”вҖ”дҪҝеҫ—дҪҝз”ЁжҹҗдёӘеҠҹиғҪзҡ„ејҖеҸ‘иҖ…дёҚйңҖиҰҒе…іжіЁеҗ„з§Қдҫқиө–еә“зҡ„еӨ„зҗҶпјҢдёҚйңҖиҰҒе…·дҪ“зҡ„й…ҚзҪ®дҝЎжҒҜпјҢ з”ұSpring BootиҮӘеҠЁйҖҡиҝҮclasspathи·Ҝеҫ„дёӢзҡ„зұ»еҸ‘зҺ°йңҖиҰҒзҡ„BeanпјҢе№¶з»Үе…Ҙзӣёеә”зҡ„BeanгҖӮдёҫдёӘж —еӯҗпјҢspring-boot-starter-jdbcиҝҷдёӘstarterзҡ„еӯҳеңЁпјҢ дҪҝеҫ—жҲ‘д»¬еҸӘйңҖиҰҒеңЁBookPubApplicationдёӢз”Ё@Autowiredеј•е…ҘDataSourceзҡ„beanе°ұеҸҜд»ҘпјҢSpring BootдјҡиҮӘеҠЁеҲӣе»әDataSourceзҡ„е®һдҫӢгҖӮ

жң¬зҜҮе°ҶйҖҡиҝҮдёҖдёӘз®ҖеҚ•зҡ„дҫӢеӯҗжқҘжј”зӨәеҰӮдҪ•зј–еҶҷиҮӘе·ұзҡ„starterгҖӮ

иҝҷйҮҢи®ІдёҖдёӢжҲ‘д»¬зҡ„StarterиҰҒе®һзҺ°зҡ„еҠҹиғҪпјҢеҫҲз®ҖеҚ•пјҢжҸҗдҫӣдёҖдёӘServiceпјҢеҢ…еҗ«дёҖдёӘиғҪеӨҹе°Ҷеӯ—з¬ҰдёІеҠ дёҠеүҚеҗҺзјҖзҡ„ж–№жі•String wrap(String word)гҖӮ иҖҢе…·дҪ“зҡ„еүҚзјҖе’ҢеҗҺзјҖжҳҜйҖҡиҝҮиҜ»еҸ–SpringBootй…ҚзҪ®ж–Үд»¶application.ymlиҖҢиҺ·еҸ–зҡ„гҖӮ

ж·»еҠ mavenдҫқиө–

з¬¬дёҖжӯҘеҪ“з„¶жҳҜеҲӣе»әдёҖдёӘmavenе·ҘзЁӢпјҢж·»еҠ SpringBootзҡ„иҮӘеҠЁй…ҚзҪ®зҡ„дҫқиө–пјҡ

жіЁж„Ҹе…¶дёӯ spring-boot-configuration-processor зҡ„дҪңз”ЁжҳҜзј–иҜ‘ж—¶з”ҹжҲҗspring-configuration-metadata.jsonпјҢ жӯӨж–Үд»¶дё»иҰҒз»ҷIDEдҪҝз”ЁпјҢз”ЁдәҺжҸҗзӨәдҪҝз”ЁгҖӮеҰӮеңЁintellij ideaдёӯпјҢеҪ“й…ҚзҪ®жӯӨjarзӣёе…ій…ҚзҪ®еұһжҖ§еңЁapplication.ymlпјҢ дҪ еҸҜд»Ҙз”Ёctlr+йј ж Үе·Ұй”®пјҢIDEдјҡи·іиҪ¬еҲ°дҪ й…ҚзҪ®жӯӨеұһжҖ§зҡ„зұ»дёӯгҖӮ

иҝҷйҮҢиҜҙдёӢartifactIdзҡ„е‘ҪеҗҚй—®йўҳпјҢSpring е®ҳж–№ StarterйҖҡеёёе‘ҪеҗҚдёәspring-boot-starter-{name} еҰӮ spring-boot-starter-webгҖӮ

Springе®ҳж–№е»әи®®йқһе®ҳж–№Starterе‘ҪеҗҚеә”йҒөеҫӘ{name}-spring-boot-starterзҡ„ж јејҸгҖӮ
